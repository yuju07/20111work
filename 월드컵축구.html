우루과이가 1골을 득점하고 한국이 0골을 득점할 확률

우루과이 득점 확률 (λ = 1.5)
  • 0골: ( P(X = 0) = \frac{1.5^0 e^{-1.5}}{0!} = 0.2231 )
  • 1골: ( P(X = 1) = \frac{1.5^1 e^{-1.5}}{1!} = 0.3347 )
  • 2골: ( P(X = 2) = \frac{1.5^2 e^{-1.5}}{2!} = 0.2510 )
  • 3골: ( P(X = 3) = \frac{1.5^3 e^{-1.5}}{3!} = 0.1255 )
    한국 득점 확률 (λ = 1.2)
  • 0골: ( P(X = 0) = \frac{1.2^0 e^{-1.2}}{0!} = 0.3012 )
  • 1골: ( P(X = 1) = \frac{1.2^1 e^{-1.2}}{1!} = 0.3614 )
  • 2골: ( P(X = 2) = \frac{1.2^2 e^{-1.2}}{2!} = 0.2168 )
  • 3골: ( P(X = 3) = \frac{1.2^3 e^{-1.2}}{3!} = 0.0867 )
